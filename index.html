Coding Projects

Other than this website, I have been involved with a few other projects.

Discord Music Bot


Github Link
The premise of this project was to create a robot using python and the Discord API to be able to listen to music together while talking to friends. The robot was self hosted and could be started from any computer that had the bot's personal ID. We used PostgreSQL to store things like the music name, artist, genre, and more so music could be stored in one location and played based off a range of specifications. If the music was not in the database already then it would download it off the internet using youtubedl (youtubedownload).

Narrative Video Game


Github Link
This project was a team based project that was done during Senior Design. The idea of this was to create a immersive, narrative video game that people could use to distract themselves during Covid-19 lockdown. We hand modeled all of the objects and environments with Blender and seamlessly implemented it into Unity where we put the world together and created the scripts to play the game. Unity uses C# for it's scripts and we had many different scripts for things like gravity and picking things up.
